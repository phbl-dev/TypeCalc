    Class CachedArrayFormula

    CachedArrayFormula (CAF) is used to calculate the results of a formula that outputs the result to multiple cells.

    Instead of recalculating the formula for each output cell individually, CAF evaluates it once and stores the result, making the rest of the dependent cells refer to this cached result.

    For example, the function FREQUENCY outputs its result to three different cells so CAF calculates the results once and store the results in the cached value of the formula.
