    Class NumberValue

    NumberValue is the designated way to handle numbers in ExcellentTS The implementation draws heavy inspiration from CoreCalc, and is heavily simplified at the same time with TypeScript number type.

    See

    • Make - Creates a new NumberValue instance and returns the instance to the user. This is the correct way to create a new NumberValue object
    • ToNumber - The way to convert a NumberValue type to an object
    • FromNumber - The way to convert an object to a NumberValue type
