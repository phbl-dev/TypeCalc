    Interface ICstNodeVisitor<IN, OUT>

    Interface representing a Concrete Syntax Tree (CST) node visitor. This interface is used to traverse and process various nodes within a CST by implementing methods corresponding to each type of node.

    interface ICstNodeVisitor<IN, OUT> {

    }
